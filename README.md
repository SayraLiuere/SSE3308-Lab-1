# SSE3308-Lab-1

SSE3308 Individual Assignment 1
Semester 2 2024/2025


BUILDING A SIMPLE PERSONAL PROFILE HTML PAGE
Duration: 2 Weeks



This is an Individual Lab Assignment (10%) for you to design and develop an attractive personal website using HTML and CSS with your creativity. 

Specifications:


1.	Plan to design your personal website with good layout and flow of useful pages.
a.	Minimum number of pages 6 with useful content including different multimedia elements.
b.	You must have at least one example of the following tags: span, p, div, h1, and ul
c.	You must have at least one table.
d.	You must have at least one html absolute and relative link.
e.	You must have at least one image right in the middle of the test like a character, and make it clickable.
f.	You must have some bold text in the document and some italics text in the document.


2.	Your personal profile page will be well-formed HTML5, and indicate that it is in the UTF-8 character set.

<!DOCTYPE html>
<html>
<head>
  <title>JaneStudent Matric# Individual Project</title>
  <meta charset="UTF-8">
</head>
<body>
...
</body>
</html>

https://validator.w3.org/nu/
